# Socium

Backend for an anonymous chat app application made with React Native.

### Tech stack
- GraphQL (API)
- Websocket (WebRTC Signaling, Chat status)
- Prisma (Postgres)
- Bull (Queues)
- Redis (Caching, Queues)
